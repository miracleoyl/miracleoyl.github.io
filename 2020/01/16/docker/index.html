docker

Installation

https://www.runoob.com/docker/ubuntu-docker-install.html

Basic commands

docker images 查看所有本地镜像
docker ps -a 查看所有本地创建的容器
docker rm 删掉一个本地容器
docker rmi : 删除本地一个或多少镜像

build and push

docker tag c8917e372922 cd-docker-registry.successfactors.com:5000/bizx-database/oracle-xe-rmda:trunk.weekly
docker push cd-docker-registry.successfactors.com:5000/bizx-database/oracle-xe-rmda:trunk.weekly

Save image

docker commit <containerID> <your imagename>
docker save -o <your saved name>.tar <image>

Load image

Docker load -i <your saved name>.tar

Troubleshooting

docker 解决 x509: certificate signed by unknown authority

/etc/docker/daemon.json添加如下配置
{
“insecure-registries”: [“registry.xxx.xxx.cn”]
}
